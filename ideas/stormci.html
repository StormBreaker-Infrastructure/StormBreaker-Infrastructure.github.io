StormBreakerCI App

Mentor :- Saalim


StormCI Application is written in Kotlin and is designed in XML layouts where we provide information about the device software (kernel-version and device codename).
Information about the supported devices that can execute our software and the team members that worked on it are displayed by the application using retrofit libraries and recyclerView.

Ideas
